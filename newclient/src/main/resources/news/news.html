Welcome to the Spaced alpha! (A4)


This is the content downloader for the Spaced alpha client. For bug reports and feature suggestions, please go to http://forums.fearlessgames.se and log in using your Fearless Games account. If you don't have an account you can register at http://fame.fearlessgames.se

Changes

A summary of the latest changes:
  • Trade interactions with players and vendors are now initiated by right clicking on the entity
  • Shields now regenerate much faster outside of combat. The regeneration is also based on max shield so it will scale with gear
  • Abilities can now have requirements to be used, e.g. ranged attacks now requires a ranged weapon to be performed
  • The wallet has moved into the inventory
  • The graphics settings can now be configured using the settings dialogue
  • Speed can now be affected by buffs such as run speed, snares and roots
  • The base damage of attacks are now displayed in the tooltip
